# Aeonaxy Assignment 
This is the assignment task for Aeonaxy Technologies Pvt Ltd

This a full-stack project [ frontend+backend ] , made using `ReactJS`+`Vite` for frontend, `NodeJS` for creating the backend. 
The web servers are created using ExpressJS, and I have used `Neon`'s free tier plan for using an online `PostgreSQL` database for storing the signup information ,
i.e, Name, Username, Email, Password [hased using bcrypt] , image URL, location. 

For generating the 
image URL , I have used the free tier of `Cloudinary` as mentioned in the assignment. After the user signs up and reaches the /verify-mail page, a mail is sent to him. 

For sending the mail, I have used the free version of `Resend`.
> Since I do not have any domain, hence currently I can only send the emails to myself, however if given a valid custom domain, this application can
send the email to the intended client. 

## Tech Stacks used : 
- ReactJS+Vite
- NodeJS
- ExpressJS
- Neon + PostgreSQL
- TailwindCSS

## Deployed on :
